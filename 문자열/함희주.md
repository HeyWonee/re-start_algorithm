# [시저 암호](https://school.programmers.co.kr/learn/courses/30/lessons/12926)

## CODE

```python
def solution(s, n):
    answer = ''
    for ch in s:
        if not ch.isalpha():
            answer += ch
            continue
        tmp = ord(ch) + n
        if ch.isupper():
            if tmp > ord('Z'):
                tmp += ord('A') - ord('Z') - 1
        else:
            if tmp > ord('z'):
                tmp += ord('a') - ord('z') - 1
        answer += chr(tmp)
    return answer

```

# [이상한 문자 만들기](https://school.programmers.co.kr/learn/courses/30/lessons/12930)

## CODE

```python
def solution(s):
    answer = ''
    words = s.split(' ')
    arr = []
    for word in words:
        tmp = ''
        for i, ch in enumerate(word):
            if i % 2 == 0:
                tmp += ch.upper()
            else:
                tmp += ch.lower()
        arr.append(tmp)
    answer = ' '.join(arr)
    return answer

```

# [튜플](https://school.programmers.co.kr/learn/courses/30/lessons/64065)

```python
def solution(s):
    s = s[2:-2]
    answer = []
    arr = s.split('},{')
    arr.sort(key=len)
    for x in arr:
        for num in map(int, x.split(',')):
            if num not in answer:
                answer.append(num)
    return answer

```
